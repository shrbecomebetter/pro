## 🚀 三秒快速上手
### 环境准备

# 1. 安装Python环境（需3.10+）

# 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 🛠️ 配置向导
1. 复制`.env.example`并重命名为`.env`
2. 填写您的硅流平台密钥：
```env:.env
OPENAI_API_BASE = "https://api.siliconflow.cn/v1/"
OPENAI_API_KEY = "你的专属密钥"
```


## ⚠️ 注意事项
1. 请妥善保管API密钥，不要上传到公开仓库
2. 调用频率限制：硅流平台默认每分钟60次请求
3. 遇到问题请先检查网络连接和密钥有效性
